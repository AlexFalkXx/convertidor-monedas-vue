🪙 Convertidor de Monedas (Vue.js)

Aplicación web creada con Vue.js que permite convertir valores entre diferentes monedas utilizando tasas de cambio actualizadas.
Este proyecto sirve para practicar reactividad, componentes, comunicación entre ellos y organización de un proyecto Vue moderno.

🚀 Características

- Conversión entre varias monedas.
- Interfaz simple y clara.
- Actualización instantánea del resultado.
- Arquitectura limpia basada en componentes.

🧱 Tecnologías utilizadas

- Vue 3
- JavaScript
- HTML5
- CSS3
- Vite

▶️ Cómo ejecutar el proyecto localmente

Abrí una terminal dentro de tu carpeta del proyecto:

Copiar código
C:\Users\tu_usuario\source\repos\Vue\vue-convertidor
Reemplazá tu_usuario por el nombre de usuario de tu PC.

Instalar dependencias:

Copiar código

npm install


Ejecutar el servidor local:

Copiar código

npm run dev

Acceder desde el navegador:

Copiar código
http://localhost:5173/
📦 Cómo generar build para producción


Copiar código

npm run build

Los archivos optimizados quedarán en la carpeta dist/.
